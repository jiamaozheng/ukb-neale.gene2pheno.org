How To Explore UKB Neale MetaXcan Data Repository Explorer

The UKB Neale MetaXcan Data Repository hosts MetaXcan results which are analyzed from over 2000 phenotype GWAS results provided by Neale Lab with the use of prediction models of expression traits across 44 human tissues (GTEx-V6p-HapMap-2016-09-08.tar.gz). This repository offers direct individual, phenotype-grouped and bulk download links, and the relevent results can be retrieved by keyword search such as phenotype name and GTEx tissue name.

Major Tools Used to Generate Data

MetaXcan : An extension of PrediXcan Method , that infers the results of PrediXcan using only summary statistics. Learn more →
